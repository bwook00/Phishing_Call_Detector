# Corpus v2 Naturalness Guide

## 핵심 원칙
- 길이를 늘릴 때는 말수만 늘리는 것이 아니라 `맥락 전개`를 추가한다.
- 자연스러움 우선순위:
  - 맥락 전환이 있는 질문/답변 교차
  - 피해자 반박, 확인, 망설임 포함
  - 사기자가 한 번에 강한 압박을 주지 않고 단계적으로 밀어붙임
- 장문은 `정보량`보다 `대화의 리듬`이 핵심이다.

## Corpus v2 길이 기준안
- short: 12~15턴
- medium: 16~19턴
- long: 21~27턴
- 총 300개 기준은 패턴별 30개를 유지한다.
  - 패턴당 분포: short 10 / medium 12 / long 8
  - 전체 분포: short 100 / medium 120 / long 80

## 패턴별 turn 추가 규칙 (실제 대화처럼 쌓기)

### 1) 기관사칭형
- 패턴 유지: 수사/기관 안내 → 계좌 확인 → 긴급성 제시 → 정지/제재 안내 → 자산 보호 행동 유도
- 추가 턴 예시:
  - 피해자 본인 이름/가입일재확인
  - 기관 절차명(온라인 민원/재확인 단계) 하나 넣기
  - 피해자 반박 후 대안 제시(“지금 이 통화가 아니면 다음 단계로 가기 어렵다”)

### 2) 고립유도형
- 패턴 유지: 장소 고정 요구 → 주변 통화 차단 → 통화 지속 유도 → 금전행동 지시
- 추가 턴 예시:
  - 피해자 귀가/이동 중인지 재확인
  - 주변에 누가 있는지 확인 대사
  - “지금은 듣기만 하고 대답만” 같은 통제형 지시 추가

### 3) 가족자녀사칭형
- 패턴 유지: 가족 위기 알림 → 급박성 → 타 계좌/중계 계좌 제안 → 심리적 압박
- 추가 턴 예시:
  - 피해자가 위조 의심을 표시하는 반박
  - 사칭자가 동선/사연 디테일 1개 보강(병원/학원/작업 현장)
  - “지금 보내지 않으면 내가 다친다” 계열 맥락 1회 추가

### 4) 납치빙자AI음성형
- 패턴 유지: 긴급 체류/납치 주장 → 위치 확인 유도 → 즉시 송금
- 추가 턴 예시:
  - 피해자 의심 시 음성 증명·위치 코드 전달 시도
  - 아이/가족 안전 점검을 빙자한 마지막 확인문
  - 대체 수단 제시 후에도 포기하지 않는 설득

### 5) 대환대출사칭형
- 패턴 유지: 기존 대출 혜택 제시 → 상환/연체 압박 → 계좌 이전 제안
- 추가 턴 예시:
  - 피해자에게 잔액/연체 시나리오를 스스로 되묻기
  - 서류/증빙명만 듣는 듯한 절차형 대화
  - “오늘 1회차 처리 안 되면 연체가 누적된다” 반복 압박

### 6) 선입금요구형
- 패턴 유지: 구매/발송 연계 → 선입금 요청 → 대체 계좌 제시
- 추가 턴 예시:
  - 피해자가 비용·영수증 상세를 물을 때 대답 지연 후 회피
  - 반품/취소 규정 하나 덧붙여 신뢰감 부여
  - 피해자 최종 확인(“맞으면 계좌 끝자리만” 등) 유도

### 7) 악성앱원격제어형
- 패턴 유지: 보안 점검 명목 → 앱 설치 → 원격권한 요청
- 추가 턴 예시:
  - 피해자의 기기 상태 확인(메모리/공간/OS 버전 언급)
  - 설치 실패/권한 오류를 만회하는 재시도 대화
  - 임시 코드를 받아 재접속하려는 절차형 대화

### 8) 카드배송사칭형
- 패턴 유지: 배송 지연 안내 → 본인 확인 → 추가 수수료/교환비 요구 → 즉시 결제
- 추가 턴 예시:
  - 피해자가 주소를 바꿔 보려는 맥락 추가
  - 택배사 단계를 한 단계 더 넣기(집중센터 접수 → 재배송)
  - 비용 영수증/문자 확인 절차를 한 번 더 제시

### 9) 법원등기우편사칭형
- 패턴 유지: 소장/등기 통보 → 미열람 제재 언급 → 링크 이동 → 사안 해결
- 추가 턴 예시:
  - 피해자 사건번호/이름 불일치 반박 장면
  - 사칭자가 증빙 자료를 “문자 확인본”으로 설명하는 변주
  - 링크 미클릭 시 대체 안내(콜센터 번호 제시)로 오히려 신뢰감 모사

### 10) 스미싱큐싱연계형
- 패턴 유지: 링크/검증 요청 → 앱/QR 유도 → 개인정보 수집
- 추가 턴 예시:
  - 피해자의 접속 실패/오류 응답 처리
  - “다른 브라우저/설정 변경” 같은 실수 대화
  - 보안 경고 문구 무시를 유도하는 정당화 한 번 추가

## 즉시 적용 규칙
- 8~10턴 미만은 재작성
- long 계열은 21턴 미만이면 2~3회 턴 블록으로 보강
- 중복 filler(어, 그게, 음, 그러니까) 제거를 generation 후점검
- 매 샘플마다 `핵심 패턴 1개만 유지`를 강제
